# আকিজ হেলথকেয়ার অর্ডার অ্যাপ

অফলাইনে চলে · কোনো সার্ভার লাগে না · লোকাল স্টোরেজে সব ডাটা সেভ হয়  
অসীম কর্মকার (ডুমুরিয়া) এর জন্য কাস্টম তৈরি করা অর্ডার ম্যানেজমেন্ট সিস্টেম

### ফিচার সমূহ
- পাসওয়ার্ড প্রটেক্টেড (536908)
- রিয়েল-টাইম কার্ট ও টোটাল হিসাব
- প্রিন্ট + হোয়াটসঅ্যাপে অর্ডার পাঠানো
- কাস্টমার ম্যানেজমেন্ট
- স্টক ম্যানেজমেন্ট (কম স্টক এলার্ট)
- অর্ডার টেমপ্লেট সেভ/লোড
- সেলস রিপোর্ট + চার্ট
- ডাটা এক্সপোর্ট/ইম্পোর্ট (JSON)

### কিভাবে ব্যবহার করবেন?
1. এই লিংকে ক্লিক করুন → https://আপনার-ইউজারনেম.github.io/akij-healthcare-order-app/
2. পাসওয়ার্ড: `536908`
3. অর্ডার নিন, প্রিন্ট করুন বা হোয়াটসঅ্যাপে পাঠান!

### অফলাইনে ব্যবহার
- একবার ওপেন করলে পরবর্তীতে ইন্টারনেট ছাড়াই চলবে
- ডাটা মুছে গেলে “সেটিংস → সব ডাটা রিসেট” করবেন না
